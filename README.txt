## Installation

Apart from normal procedure to enable drupal modules, you would have to download the library located at https://github.com/farbelous/fontawesome-iconpicker.

Please download the library from the above location and put it in libraries folder. After that unzip and rename the folder to 'fontawesome-iconpicker' (i.e. remove the version suffix). After you do it, the fontawesome-iconpicker.js file should be available at /sites/all/libraries/fontawesome-iconpicker/dist/js/fontawesome-iconpicker.js.

## Known Issues

Sometimes the jQuery module supports a different version of the Font Awesome CSS that's installed. In that case, manually update the Font Awesome CSS to match.
